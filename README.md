# Minoan-Robot-2026
🏆 MRC 2026 Line Follower Robot
Achievement: 3rd Place - Minoan Robot Sports Competition (Marathon)

📌 Overview
This project is a line-following robot built for the 2026 MRC Global Olympiad. The robot uses an optimized IR Sensors to navigate complex tracks and maintain speed during sharp turns.

📸 Demo
[📺 Click here to watch my Robot Marathon Video](./MARATHON_VIDEO.mp4)

🛠 Tech Stack
Controller: Arduino Uno

Programming Logic: PictoBlocks

Sensors: 3 IR Sensors

Motors: 2 dc motors on the back + L298N H-Bridge motor driver

💡 What I Learned
Learned how to tune PID control logic for smoother turns.

Optimized sensor sensitivity for different track lighting conditions.

Gained experience in rapid troubleshooting and hardware-software integration.
